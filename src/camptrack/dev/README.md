# Development Utilities

> [!NOTE]
> This assumes you have installed the package via `python -m pip install -e .`
> when in the project root.

Available commands:
```bash
Usage: camptrack-dev <command>
Commands:
    delete    - Delete the database
    reset     - Delete then reset the database to its default state (e.g. default users)
    path      - Get the path to the SQLite database
```
